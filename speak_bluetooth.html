接続図¶

JY-MCU と Arduino はクロス配線となるように接続する。 JY-MCU の RX 端子を Arduino デジタルピン 1番(TX) につなぎ、JY-MCU の TX 端子を Arduino デジタルピン 0 番 (RX) につなぐ。

ソースコード¶

Arduino に以下のコードをアップロードしておく。

void setup() {
  Serial.begin(9600);
  }

  void loop() {
    delay(1000);
      Serial.println("hello world!");
  }

Mac からデバイスに接続する¶

  1. システム環境設定 > Bluetooth を開く。
  2. デバイスに HC-06 と表示されるので、ペアリングを押す。
  3. ペアリングできませんでしたと表示されるので、オプションを選択してコード欄に 1234 と入力して再度ペアリングを押す。
  4. HC-06 接続済みとなる。
